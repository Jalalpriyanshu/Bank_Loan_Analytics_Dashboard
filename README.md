# 📊 Bank Loan Analytics Dashboard

## 📌 Project Overview

The **Bank Loan Analytics Dashboard** is an interactive Power BI project designed to analyze and monitor bank loan performance.
It provides valuable insights into loan applications, funded amounts, repayments, interest rates, debt-to-income (DTI) ratios, and overall loan quality.
The dashboard helps stakeholders make data-driven decisions by identifying lending trends, monitoring portfolio performance, and evaluating loan risk through interactive visualizations.

## 🚀 Dashboard Features

### 📈 Summary Dashboard
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI Ratio
- Good Loan vs Bad Loan Analysis
- Funded Amount vs Repayment Analysis
- Loan Status Comparison
- Interest Rate Analysis
- DTI Ratio Analysis

### 🌍 Overview Dashboard
- Monthly Loan Application Trends
- Loan Applications by State
- Loan Purpose Analysis
- Home Ownership Analysis
- Loan Term Distribution
- Geographic Loan Insights

---

## 📊 Key Performance Indicators (KPIs)

| KPI | Description |
| Total Loan Applications | Total number of loan applications received |
| Total Funded Amount | Total amount funded by the bank |
| Total Amount Received | Total repayments received from borrowers |
| Average Interest Rate | Average interest rate across all loans |
| Average DTI Ratio | Average Debt-to-Income ratio of borrowers |
